\doxysubsection{Detailed Description}
This file contains the function for a task that records the motor shaft position of both motors and converts the ticks into angular position which is then used for to generate the error signal in the conversiont task This task includes the encoder driver from the ME 507 support Git hub, found here\+: \href{https://github.com/spluttflob/ME507-Support.git}{\texttt{ https\+://github.\+com/spluttflob/\+ME507-\/\+Support.\+git}}

Task which reads from the motor encoders to track position for closed-\/loop position control. 

This task monitors the angular position of the motors, converts the position into degrees, and writes the encoder values into two shared queues for use by the conversion task.
